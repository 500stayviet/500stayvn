# i18n 수동 스모크 (KO / VI / EN / JA / ZH)

앱 상단 또는 설정에서 언어를 바꾼 뒤, **각 언어마다** 아래를 한 번씩 수행합니다. 빈 문자열, 키 그대로 노출, 의도와 다른 단일 언어(예: 한국어만) 고정이 없는지 확인합니다.

1. **검색**: 메인 검색·목록·필터 라벨이 현재 언어로 보이는지.
2. **예약**: 날짜 선택, 게스트 수, 결제/요약 문구가 현재 언어인지.
3. **KYC**: 단계 제목·버튼·오류 메시지가 현재 언어인지.
4. **정산·출금**: 정산/출금 화면의 표·버튼·상태 문구가 현재 언어인지.
5. **약관** (`/terms`): 전체 스크롤 — 본문이 해당 언어 원문이고, SEO용 기본 메타는 영문이어도 본문은 언어와 일치하는지.
6. **개인정보** (`/privacy`): 동일하게 스크롤, 계정 삭제 링크 라벨·연락처 라벨 확인.
7. **관리자(해당 시)**: 관리자 계정 저장 오류 시 API 코드가 아닌 현재 언어 메시지로 표시되는지.

기록 예: 날짜, 테스터, 언어별 Pass/Fail, 스크린샷 또는 이슈 링크.
